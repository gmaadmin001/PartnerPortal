# Wireframe — Design Reference Only

The files in this folder (`login.html`, `register.html`, `dashboard.html`) are **design wireframes** used to prototype the Partner Portal UI.

- **Not for production use.** These are standalone HTML files using CDN dependencies and hardcoded config values.
- **Not the final implementation.** The production app is built in Next.js under `src/`. These wireframes exist only to validate layouts, flows, and visual design before implementation.
- **May be out of sync** with the production codebase at any time.
